# 前端 UI 开发总览

本目录用于记录 Kkbar 前端面板的 UI、交互、状态管理和样式系统。

## 1. 阅读顺序

建议按以下顺序阅读：

1. [系统概述](#系统概述)
2. [响应式布局](#响应式布局)
3. [样式与动画系统](#样式与动画系统)
4. [交互模式设计](#交互模式设计)
5. [设置面板设计](#设置面板设计)
6. [自适应方案](#自适应方案)
7. [开发指南](#开发指南)

## 2. 这一部分讲什么

- 面板布局如何组织
- 工具栏、分类、按钮区如何协作
- 主题切换与局部样式如何落地
- 不同窗口尺寸和布局模式下如何自适应
- 交互模式如何影响状态、提示和执行逻辑

## 3. 关键文件

- `src/js/main/main.vue`：主面板入口和执行逻辑
- `src/js/main/components/`：设置、对话框、日志、动作选择等组件
- `src/js/main/store/kbar.ts`：核心状态与持久化
- `src/js/main/main.ts`：主题变量与启动初始化

## 4. 维护建议

- UI 改动先看 `main.vue`，再看对应组件和 store
- 任何影响布局的改动，都要同步检查响应式章节
- 任何影响执行提示或交互提示的改动，都要同步检查交互模式章节
- 任何影响主题变量的改动，都要同步检查样式与动画章节

## 5. 推荐关注点

- `showActionAlerts` 是否仍符合当前交互预期
- `showToolbar`、`showButtons` 等开关是否影响默认可见性
- `layoutMode`、`listColumns` 是否与文档保持一致
- 主题变量是否覆盖到暗色和亮色两套视觉
