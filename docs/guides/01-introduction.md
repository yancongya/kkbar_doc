# 简介

## Kkbar 是什么

![首页](/assets/images/intro.png)

Kkbar 是一个 Adobe CEP 扩展工具栏，基于 KBAR 扩展项目的理念和思路进行升级开发。它提供一个可自定义的按钮面板，让你将常用的脚本、预设、效果、表达式、Shell 命令、剪贴板等操作集中管理，只需点击按钮即可快速执行。

:::tip 免费版与付费版
免费版和付费版核心功能完全一致，付费版额外移除启动弹窗、解锁云备份功能。详见 [免费版与付费版](./04-free-vs-pro.md)。
:::

## 支持的应用程序

:::warning 版本要求
当前在 **Adobe After Effects CC 2020 及以上版本**测试通过，其他版本理论上兼容，但未经完整验证。
:::

## 核心特性

- 🖥️ **多面板支持**：一个扩展三个不同的面板，可同时打开使用
- ⚡ **丰富的动作类型**：支持 JSX 脚本、预设、效果、表达式、剪贴板、Shell 命令等 9 种动作类型
- 🎨 **高度自定义**：随意控制按钮触发的快捷键和触发方式，UI 界面完全自定义，多个主题切换
- 🎯 **完善的图标系统**：内置图标、FontAwesome 图标、Bootstrap Icons 图标、自定义图片，四种方式自由搭配
- 💾 **便捷的数据管理**：数据完全本地化，配置可导入导出，方便分享和备份。也可使用 Git 仓库进行云备份处理（高级功能）
- 📋 **完整的日志记录**：打开日志面板查看执行记录，便于排查问题
- 💾 **兼容 KBar 数据库**：支持导入 KBar 格式的 `.kbar` 文件，自动迁移分类、按钮、脚本、预设等数据，方便从 KBar 无缝切换
