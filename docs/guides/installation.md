# 安装

## 系统要求

- Adobe After Effects CC 2020+ 或 Photoshop CC 2020+
- Windows 10+ / macOS 10.15+

## 安装步骤

1. 下载最新版本的 `.zxp` 文件
2. 使用 [ZXP Installer](https://aescripts.com/learn/zxp-installer/) 或 [Anastasiy Extension Manager](https://install.anastasiy.com/) 安装
3. 重启 Adobe 软件
4. 在菜单栏选择 **窗口 > 扩展 > Kkbar**

## 开发环境

如需从源码构建：

```bash
git clone https://github.com/your-org/kkbar.git
cd kkbar
yarn
yarn build
yarn symlink
```
