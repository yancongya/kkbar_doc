# 主题与外观

## 🎨 UI 设计理念

![首页](/assets/images/uisettings.webp)

**简约至上**：

- **无文案按钮**：工具栏按钮使用纯图标设计，节省空间，视觉更简洁
- **渐进式披露**：复杂功能面板默认折叠，按需展开，避免信息过载
- **即时反馈**：所有操作都有视觉反馈（悬停、点击、拖拽）

**现代化交互**：

- **响应式设计**：面板根据窗口宽度自动调整布局
  - ≥ 600px：正常布局
  - 480-599px：紧凑布局
  - < 480px：最小布局
- **动态尺寸**：图标大小和间距实时可调，拖动滑块时面板透明预览
- **快捷键系统**：支持 Ctrl/Alt/Shift 组合，一个按钮可触发多种动作

| 主题 | 说明 |
|------|------|

## 图标设置

### 图标大小

拖动滑块调整按钮图标大小（10-50）。

拖动大小或间距滑块时，设置面板会自动变为透明，方便你直接预览主页按钮的实际效果。松开滑块后面板恢复正常。

### 图标间距

拖动滑块调整按钮之间的间距（0-30）。

## 导航栏

导航栏位于面板顶部，包含以下固定按钮：

| 按钮 | 功能 |
|------|------|
| 分类下拉 | 切换当前显示的分类 |
| 日志 | 打开日志面板，查看执行记录 |
| 切换视图 | 卡片网格 ↔ 单列列表切换 |
| 关于 | 打开关于页面（仅免费版显示） |
| 设置 | 打开设置面板 |

可在设置中开启或关闭导航栏显示。

## 分类切换器

首页按钮网格的固定位置有一个分类切换器按钮，显示当前分类的图标。

### 点击操作

| 操作 | 效果 |
|------|------|
| 点击 | 切换到下一个分类 |
| Ctrl + 点击 | 打开设置面板 |
| Shift + 点击 | 打开日志面板 |
| Alt + 点击 | 打开关于页面 |
| Ctrl + Shift + 点击 | 切换视图模式（卡片/列表） |

### 长按弹出菜单

**长按**分类切换器按钮（300ms）会弹出快捷操作菜单，包含：

- 切换分类
- 打开设置
- 打开关于
- 打开日志
- 切换视图

---

## 长按按钮菜单

首页中，**长按任意按钮**（300ms）会弹出该按钮下所有快捷动作的菜单，方便单手操作。

### 两种显示模式

| 模式 | 说明 | 切换方式 |
|------|------|---------|
| 面板模式（默认） | 居中垂直列表，显示动作名称和快捷键 | 设置中关闭"环形菜单" |
| 环形菜单模式 | 图标围绕按钮呈环形排列，鼠标移动选中 | 设置中开启"环形菜单" |

在设置面板的开关区域，可以通过 **环形菜单** 开关切换这两种模式。

## 视图模式

| 模式 | 说明 |
|------|------|
| 卡片模式（默认） | 按钮以网格形式排列 |
| 单列列表模式 | 按钮以单列列表形式排列，显示按钮名称 |

切换方式：

- 点击导航栏的 **切换视图** 按钮
- Ctrl + Shift + 点击分类切换器按钮

## 其他设置

| 开关 | 说明 |
|------|------|
