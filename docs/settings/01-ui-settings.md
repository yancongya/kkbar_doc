# 01. 界面设置

Kkbar 采用现代化、简约化的界面设计理念，在保证功能完整的同时尽可能减少不必要的视觉干扰，让用户专注于内容本身。

## UI 设计理念

### 简约至上

- **无文案按钮**：工具栏按钮使用纯图标设计，节省空间，视觉更简洁
- **渐进式披露**：复杂功能面板默认折叠，按需展开，避免信息过载
- **即时反馈**：所有操作都有视觉反馈（悬停、点击、拖拽）

### 现代化交互

- **响应式设计**：面板根据窗口宽度自动调整布局
  - ≥ 600px：正常布局
  - 480-599px：紧凑布局
  - < 480px：最小布局
- **动态尺寸**：使用 CSS 变量驱动，图标大小实时可调
- **快捷键系统**：支持 Ctrl/Alt/Shift 组合，一个按钮可触发多种动作

### 灵活多变

- **按钮透明模式**：仅显示图标，进一步节省空间
- **Tab Menu 模式**：长按按钮显示环形小图标菜单，适合触控操作
- **模式切换**：卡片模式 ↔ 列表模式一键切换

### 极简导航

- **隐藏导航栏**：可完全隐藏顶部导航栏，仅保留按钮区域
- **纯按钮操作**：所有功能通过按钮交互，无需传统菜单
- **分类切换**：点击分类的第一个按钮即可快速切换分类
- **右键操作**：在按钮上右键点击可直接打开设置对话框

## 主题切换

Kkbar 提供多种主题供选择：

| 主题名称 | 说明 |
|----------|------|
| AE Native | 适配 After Effects 的原生深色主题 |
| 深色专业 | 深色专业风格 |
| 浅色现代 | 浅色现代风格 |
| 黑曜石 | 黑曜石风格暗色主题 |
| Nord | Nord 配色风格的冷色调主题 |

切换方法：设置面板 > 主题选择

## 图标设置

### 图标大小

拖动滑块调整按钮图标大小（40px - 140px）。

### 图标间距

调整按钮之间的间距（0px - 30px）。

| 模式 | 说明 |
|------|------|
| 换行 | 按钮自动换行排列 |
| 单行 | 按钮单行水平排列 |
| 垂直 | 按钮垂直排列 |

## 工具提示

开启后，鼠标悬停在按钮上会显示按钮名称。

可设置是否显示快捷键提示（Ctrl、Shift、Alt 等修饰键）。

可在设置中开启或关闭。

---

## 显示选项

- 显示/隐藏工具栏
- 显示/隐藏按钮
- 按钮背景透明度调节（透明模式）
- 动作执行完成后显示提示

### 透明模式

开启按钮透明模式后，按钮仅显示图标，背景变为透明：

- 适合图标较小的场景
- 让界面更加简洁
- 节省视觉空间

---

## 核心特点

- **多主题支持**：5 种内置主题，满足不同审美
- **图标大小可调**：40-140px 范围可调
- **布局灵活**：支持换行、单行、垂直三种模式
- **工具提示**：显示按钮名称和快捷键
- **按钮透明模式**：仅显示图标，节省空间
- **Tab Menu 模式**：长按按钮显示环形小图标菜单
- **极简导航**：可隐藏导航栏，纯按钮操作，通过分类首按钮切换分类

## 适用场景

- 根据个人审美选择主题
- 调整图标大小适应不同屏幕
- 切换布局模式适配工作流
- 开启工具提示帮助记忆快捷键
