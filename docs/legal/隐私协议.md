# Kkbar 隐私协议

更新日期：2026年5月18日

本隐私协议适用于 Kkbar（以下简称“本软件”）及其相关功能、文档和服务。请在使用本软件前仔细阅读并理解本协议内容。您使用或继续使用本软件，即表示您已阅读、理解并同意本协议全部内容。

## 1. 信息收集范围

本软件以本地运行和本地存储为主。通常情况下，本软件不会主动收集与您身份直接相关的个人信息。

在您使用本软件过程中，可能涉及以下信息：

- 本地配置数据：如按钮、分类、主题、布局、图标设置、窗口状态等
- 本地内容数据：如脚本、预设路径、表达式内容、菜单命令、剪贴板相关内容
- 日志信息：如操作记录、错误信息、导入导出记录、调试信息
- 设备与运行环境信息：如操作系统信息、Adobe 应用环境信息、版本号、面板运行状态
- 网络请求所需信息：如访问 GitHub、Gist、更新检查、统计上报时产生的必要请求信息

## 2. 信息收集方式

本软件获取信息的方式包括：

- 您在界面中主动输入、导入、导出或保存的数据
- 本软件在本地自动生成和记录的配置、缓存和日志
- 您主动启用的网络功能所产生的请求和响应数据
- 您与第三方服务交互时产生的数据，例如 GitHub、Gist 等

## 3. 信息使用目的

本软件收集或处理信息，仅用于以下目的：

- 提供、维护和改进软件功能
- 保存您的个性化配置和使用偏好
- 执行您发起的脚本、预设、效果、表达式或菜单命令
- 处理导入、导出、备份、恢复等操作
- 记录日志以便排查问题、定位故障和优化体验
- 完成您主动启用的云同步、授权登录或更新检查等功能

## 4. 信息存储与保留

- 大部分配置和使用数据默认存储在您的本地设备中
- 您可以通过软件提供的导入、导出、清除或重置功能管理相关数据
- 日志和缓存的保留时间取决于您的使用情况及软件版本实现
- 当您卸载软件或手动清除本地存储后，相关本地数据可能被移除，但已同步到第三方服务的数据仍受第三方服务规则约束

## 5. 第三方服务

本软件可能与第三方服务发生交互，例如：

- GitHub 登录与授权
- Gist 云备份或云同步
- 外部更新检查
- 外部链接跳转

上述第三方服务由其各自的服务提供方独立运营，其隐私政策和服务条款不受本软件控制。请您在使用相关功能前自行阅读并理解第三方的规则。

## 6. 权限与访问

本软件的部分功能可能需要访问以下内容：

- 本地文件和目录，用于读取脚本、预设、图标或导出配置
- 剪贴板，用于复制文本、路径或相关内容
- 网络连接，用于执行您主动触发的在线功能
- Adobe 应用环境，用于执行与宿主程序相关的操作

本软件仅在功能需要时访问相应内容，不会在未授权或无必要的情况下持续访问。

## 7. 数据共享

除以下情形外，本软件不会主动向第三方共享您的本地数据：

- 您主动使用第三方服务或跳转外部链接
- 为完成您主动发起的云同步、授权登录或更新检查等功能
- 法律法规要求、行政机关或司法机关依法要求
- 为保护本软件、用户或公众的合法权益而有必要披露

## 8. 用户控制与删除

您可以通过以下方式控制相关数据：

- 在软件中修改或删除本地配置
- 清除本地存储、缓存或浏览器/CEP 侧数据
- 撤销 GitHub 授权或清理第三方服务中的云端内容
- 停用相关网络功能，避免产生额外的在线交互

如您删除本地数据，某些功能的历史状态可能无法恢复。

## 9. 未成年人保护

如您是未成年人，请在监护人指导下使用本软件。监护人应协助您理解本协议并指导您正确使用相关功能。

## 10. 协议变更

本软件可能根据产品功能、法律法规或运营需要对本协议进行更新。更新后的协议将在适当位置公布或随软件版本一并提供。若您继续使用本软件，即视为接受更新后的协议内容。

## 11. 联系方式

如您对本隐私协议有任何疑问、意见或建议，请通过本软件提供的联系方式、项目主页或开发者公开渠道与我们联系。
