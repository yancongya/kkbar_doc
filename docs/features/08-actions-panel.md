# 调用打开扩展面板

快速打开本地已安装的 CEP 扩展面板，无需在 AE 菜单中查找。

## 使用方法

1. 在设置面板中添加或编辑按钮
2. 设置按钮名称和图标
3. 选择动作类型为 **面板**
4. 点击刷新按钮扫描本地已安装的 CEP 面板
5. 在列表中搜索并选择要打开的面板
6. 点击左上角保存并关闭设置面板

## 核心特点

- **混合 3 策略扫描**：并行执行 API、常用目录、系统目录三种扫描策略，确保覆盖率最大化
- **高性能**：三路并发执行，自动去重优选，100+ 面板扫描仅需 60-100ms
- **日志集成**：扫描过程详细记录到日志面板，支持搜索和调试
- **右键测试**：右键点击列表项即可直接打开面板，快速验证效果
- **智能回显**：编辑已保存配置时自动匹配并显示面板名称，无需重新扫描

:::warning 执行前提
- 目标面板已在 AE 中安装（作为 CEP 扩展）
- 如果没有扫描到任何面板，请检查 AE 是否正确安装了扩展
:::

## 适用场景

- 快速切换到其他 CEP 面板（如 Kkbar 工具箱、其他扩展）
- 一键打开常用的 AE 辅助面板
- 替代 AE 菜单中的「窗口 > 扩展」查找过程
