# 预设应用 (.ffx)

应用 After Effects 动画预设（.ffx）到选中的图层。
![首页](/assets/images/presets0.png)

## 使用方法

1. 在设置面板中添加或编辑按钮
2. 设置按钮名称和图标
3. 选择动作类型为 **预设**
4. 点击文件夹图标选择预设文件（.ffx），或手动输入文件路径
5. 点击左上角保存并关闭设置面板
   ![首页](/assets/images/presets.png)

## 核心特点

- **批量应用**：支持同时将预设应用到多个选中的图层
- **文件选择**：通过系统文件选择器选择 `.ffx` 预设文件
- **UndoGroup 支持**：自动包裹在 `app.beginUndoGroup` 中，支持 AE 撤销操作
- **错误提示**：预设应用失败时显示明确的错误信息

:::warning 执行前提

- 当前 AE 中有打开的活动合成
- 已选中一个或多个图层
- 预设文件存在且有效
- 如果没有活动合成，运行时会提示"未找到活动合成"
- 如果没有选中图层，运行时会提示"请先选择一个或多个图层"
  :::

## 适用场景

- 批量添加动画效果
- 快速应用转场特效
- 为多个图层统一样式
- 复用常用动画组合
