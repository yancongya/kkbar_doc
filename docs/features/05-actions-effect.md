# 内置效果调用 (Effects)

应用 After Effects 内置或第三方效果到选中的图层。
![首页](/assets/images/eefect.png)

## 使用方法

![首页](/assets/images/effect.webp)

1. 在设置面板中添加或编辑按钮
2. 设置按钮名称和图标
3. 选择动作类型为 **效果**
4. 在搜索框中输入效果名称（支持中英文、MatchName 混合搜索）
5. 点击右侧刷新按钮加载效果列表，或直接输入触发自动加载
6. 从下拉列表中选择需要的效果
7. 点击左上角保存并关闭设置面板

## 核心特点

- **实时获取**：直接从 AE 宿主获取当前安装的全部效果列表（非硬编码字典），确保版本兼容性
- **高性能**：采用"一次加载 + 前端本地搜索"架构，避免频繁跨进程通信
- **手动触发**：通过刷新按钮手动加载，不阻塞扩展启动速度
- **智能搜索**：支持中英文、MatchName 混合搜索，并按相关度打分排序

:::warning 执行前提

- 当前 AE 中有打开的活动合成
- 已选中一个或多个图层
- 如果没有选中图层，运行时会提示"请先选择一个或多个图层"
  :::

## 适用场景

- 快速添加视觉特效（如模糊、锐化、扭曲等）
- 批量处理多个图层的效果应用
- 创建噪点/纹理背景（湍流噪声等）
- 调整颜色和曝光（曲线、色阶等）
- 特效动画制作
