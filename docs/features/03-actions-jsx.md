# 脚本文件执行 (.jsx & .jsxbin)

执行本地磁盘上的 `.jsx` 或 `.jsxbin` 脚本文件，实现复杂的自动化操作。

## 使用方法

1. 在设置面板中添加或编辑按钮
2. 设置按钮名称和图标
3. 选择动作类型为 **脚本文件**
4. 点击文件夹图标选择脚本文件，或手动输入文件路径
5. 点击左上角保存并关闭设置面板

## 核心特点

- **外部文件执行**：执行本地磁盘上的 `.jsx` 或 `.jsxbin` 脚本文件
- **完整 ExtendScript 能力**：可以调用 AE 的所有脚本 API，支持复杂的自动化任务
- **错误报告**：执行失败时显示文件名、行号和详细错误消息
- **UndoGroup 支持**：自动包裹在 `app.beginUndoGroup` 中，支持 AE 撤销操作

:::warning 执行前提
- 脚本文件路径有效且文件存在
- 如果文件不存在，运行时会提示"脚本文件不存在"
:::

## 适用场景

- 批量处理素材（导入、重命名、调整）
- 自动创建合成和设置
- 自定义动画流程和预设
- 调用外部工具或脚本
