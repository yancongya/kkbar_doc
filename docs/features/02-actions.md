# 关于按钮的操作

不同快捷键的搭配决定了同一个按钮点击时执行什么操作。Kkbar 支持为同一个按钮设置多个动作，通过不同的快捷键触发不同功能。

## 快捷键系统

### 支持的快捷键组合


| 快捷键         | 说明                 |
| -------------- | -------------------- |
| (无)           | 点击                 |
| Ctrl           | 按住 Ctrl 点击       |
| Alt            | 按住 Alt 点击        |
| Shift          | 按住 Shift 点击      |
| Ctrl+Alt       | 按住 Ctrl+Alt 点击   |
| Ctrl+Shift     | 按住 Ctrl+Shift 点击 |
| Alt+Shift      | 按住 Alt+Shift 点击  |
| Ctrl+Alt+Shift | 按住全部点击         |

### 动态图标

首页中，根据当前按下的快捷键，按钮可以显示不同的图标：

- 默认图标：单击按钮时候的基本图标
- 快捷键图标：按下修饰键时，可显示与该动作关联的专属图标

这使得同一个按钮在不同触发方式下具有不同的视觉反馈。

## 快捷拖拽操作

### 主页编辑模式（长按空格）

在主页界面：

- **长按空格**：进入编辑模式
- **松开空格**：退出编辑模式

编辑模式下支持：

- 拖拽排序
- 拖动到其他按钮上交换位置
- 调整按钮大小（拖动右下角调整手柄）

右键操作：

- **对应热键模式下右键点击按钮**：打开动作设置面板，可以编辑该按钮对应热键下的动作配置

长按弹出动作面板：

在主页，除了快捷键还可以按钮上**长按**可以快速执行该按钮下的各个快捷动作，更适合单手模式：

- **长按触发**：在按钮上按住鼠标 300ms 后弹出动作面板
- **两种模式，可在设置中切换模式**：
  - 底部面板模式：垂直列表面板
  - 环形菜单模式：显示为环形小图标
- **拖拽选择**：鼠标在面板上移动时自动选中对应动作

### 设置页拖拽

在设置面板中的图标网格进行拖拽：

- **直接拖拽**：调整按钮在当前分类内的顺序
- **跨分类拖拽**：将按钮移动到其他分类
- **Ctrl+拖拽**：克隆按钮到目标分类
- **拖到底部的垃圾桶**：删除按钮
