# SVG 生成器

Kkbar 内置了文字转 SVG 图标功能，无需设计工具即可快速创建简单的文字图标。

## 使用方法

1. 在图标选择器中点击 **生成** 按钮
2. 生成器面板会展开
3. 输入文字
4. 设置文字颜色和背景颜色
5. 调整字体大小
6. 实时预览效果
7. 点击 **生成 SVG** 按钮
8. 生成的图标自动添加到图标库

---

## 生成器选项

| 选项 | 说明 |
|------|------|
| 文字 | 图标上显示的文字（1-4 个字符效果最佳） |
| 文字颜色 | 文字的前景色 |
| 背景颜色 | 背景矩形的填充颜色 |
| 背景透明 | 开启后背景变为透明 |
| 字体大小 | 控制文字在图标中的比例（20-80px） |

---

## 生成的 SVG 规格

| 属性 | 值 |
|------|-----|
| 画布尺寸 | 100×100 px（viewBox） |
| 背景形状 | 圆角矩形（圆角半径 16px） |
| 文字对齐 | 水平居中 + 垂直居中 |

---

## 预览

生成器面板展开时，图标网格中第一个位置会实时显示预览效果，方便调整参数。修改文字、颜色、大小后预览会立即更新，确认满意后再点击"生成 SVG"按钮。

## 适用场景

- 快速创建文字图标（如 "AE"、"导出"、"P"）
- 制作首字母缩写图标
- 不需要复杂图形的简洁图标
- 创建带背景色的功能标识
