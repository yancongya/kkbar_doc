# 01. 图标系统

Kkbar 提供完整的图标管理系统，支持多种图标来源，满足不同场景需求。

## 图标来源

| 来源 | 说明 | 文档 |
|------|------|------|
| 内置图标 | Kkbar 内置的 SVG 图标 | [内置图标](./02-icons-builtin.md) |
| FontAwesome | FontAwesome 4.7 图标库 | [FontAwesome 图标](./03-icons-fontawesome.md) |
| 导入图标 | 图片文件或 SVG 代码 | [导入图标](./04-icons-import.md) |
| 图标颜色 | 自定义图标颜色 | [图标颜色](./05-icons-color.md) |

## 图标选择器

在创建或编辑按钮或分类时，点击图标区域会弹出图标选择器。

### 选择器布局

```
┌─────────────────────────────────────────┐

├─────────────────────────────────────────┤
│                                         │
│  图标网格（显示所选分类的图标）           │
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐  │
│  │ A │ │ B │ │ C │ │ D │ │ E │ │ F │  │
```

### 工具栏说明

| 按钮 | 功能 |
|------|------|
| Iconfont | 打开 iconfont.cn 中国图标库（需联网） |
| FontAwesome | 打开 fontawesome.com 官方图标库（需联网） |
| FA | 显示 FontAwesome 图标 |
| BI | 显示 Bootstrap Icons 图标 |
| 内置 | 显示内置 SVG 图标 |
| 导入 | 导入本地图片文件 |
| 生成 | 文字转 SVG 图标生成器 |
| 粘贴 | 粘贴 SVG 代码 |
| 关闭 | 关闭图标选择器 |

### 搜索功能

- 支持关键词搜索
- 实时过滤显示匹配的图标
- 搜索不区分大小写

---

## 为按钮设置图标

1. 在设置面板中添加或编辑按钮
2. 点击按钮的图标区域
3. 在图标选择器中选择图标
4. 点击确认

---

## 为分类设置图标

1. 在设置面板中找到要设置图标的分类
2. 点击分类的图标区域
3. 在图标选择器中选择图标
4. 点击确认

---

## 图标大小

在图标选择器中可以调整图标显示大小：

- 拖动图标大小滑块（32px - 128px，步长 4px）
- 点击 **重置** 按钮可恢复默认大小（72px）
- 右侧显示当前尺寸数值

### 图标过滤

在图标大小控制栏右侧有四个过滤按钮：

| 按钮 | 功能 |
|------|------|
| FA | 显示/隐藏 FontAwesome 图标 |
| BI | 显示/隐藏 Bootstrap Icons 图标 |
| 内置 | 显示/隐藏内置 SVG 图标 |
| 导入 | 显示/隐藏用户导入的图标 |

点击按钮切换显示/隐藏状态，多个按钮可以同时激活组合筛选。

## 核心特点

- **多种图标来源**：支持内置 SVG、FontAwesome、Bootstrap Icons、图片文件、SVG 代码多种来源
- **实时搜索**：关键词搜索，实时过滤，搜索不区分大小写
- **图标过滤**：可按类型筛选显示（FA/BI/内置/导入）
- **大小可调**：图标大小 32px - 128px 可调
- **颜色自定义**：支持为 FA、BI、内置图标设置自定义颜色
- **离线可用**：内置图标和 FontAwesome 无需联网即可使用
- **SVG 生成器**：内置文字转 SVG 图标功能，快速创建简单图标
- **右键删除**：导入的图标支持右键删除管理

## 适用场景

- 为按钮设置功能对应的图标，便于识别
- 为分类设置专属图标，美化界面布局
- 使用自定义图片作为项目专属 Logo
- 使用文字图标快速标注（如 "AE"、"导出"）
- 统一多个按钮的图标风格，提升整体美观度
