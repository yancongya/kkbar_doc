# 内置图标

Kkbar 内置了 5 个常用 SVG 图标，无需联网即可使用。

## 使用方法

1. 在图标选择器中点击 **内置** 过滤按钮
2. 浏览图标或搜索关键词
3. 点击选择图标

## 内置图标列表

| 图标名称 | 含义 | 说明 |
|----------|------|------|
| bolt | 闪电/运行 | 代表执行、运行动作 |
| folder | 文件夹 | 代表文件夹、目录 |
| gear | 齿轮 | 代表设置、配置 |
| run | 播放 | 代表开始、运行 |
| trash | 垃圾桶 | 代表删除、清理 |

## 核心特点

- **无需联网**：图标已内置在扩展中，离线可用
- **矢量清晰**：SVG 格式，无限缩放不失真
- **可自定义颜色**：支持设置图标颜色
- **稳定可靠**：不会因外部依赖导致图标无法显示

## 适用场景

- 追求稳定性和离线使用
- 简单功能图标需求
- 不需要太复杂的图标场景
