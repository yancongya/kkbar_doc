# 导入图标

除了内置图标库，Kkbar 还支持导入自定义图片作为图标。
![首页](/assets/images/iconinput.png)

## 导入方式

### 方式一：导入图片文件

1. 在图标选择器中点击 **导入** 按钮
2. 在弹出的文件选择器中选择图片文件
3. 图标自动添加到图标库

:::tip 支持的图片格式

| 格式 | 说明 | 适用场景 |
|------|------|---------|
| PNG | 支持透明背景 | Logo、复杂图标 |
| JPG/JPEG | 有损压缩 | 照片、复杂图像 |
| SVG | 矢量图形 | 矢量图标、Logo |
| GIF | 动态图标 | 简单动画效果 |
| WebP | 现代格式 | 高压缩比图像 |

:::

### 方式二：粘贴 SVG 代码

如果已有 SVG 代码，可以直接粘贴导入：

1. 在图标选择器中点击 **粘贴** 按钮
2. 粘贴面板会展开
3. 在文本框中粘贴 SVG 代码
4. 输入图标名称（用于识别）
5. 点击 **保存 SVG** 按钮
6. 图标会出现在图标网格中

### 方式三：拖拽导入

:::tip 快捷导入
在图标选择器打开时，直接将图片文件拖拽到图标选择器窗口中，系统会自动导入。
:::

---

## 导入的图标管理

导入的图标保存在图标库中：

- 在图标选择器中点击 **导入** 标签查看所有已导入图标
- **右键点击**图标可删除
- 导入的图标在 CEP 和浏览器模式下均可使用

---

## 图标存储位置

| 环境 | 存储位置 |
|------|---------|
| CEP 模式 | `{userData}/KkbarDB/icons/` |
| 浏览器模式 | `localStorage` |

:::warning 注意

- 导入的图片文件图标**不支持**自定义颜色
- 导入的 SVG 代码如果内部已定义颜色，也无法通过颜色选择器修改
  :::

:::tip 适用场景

- 使用自己的 Logo 作为图标
- 使用项目专属的图标
- 导入现有的 SVG 素材作为图标
  :::
