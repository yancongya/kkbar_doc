# 05. 图标颜色

可以为图标设置自定义颜色，使界面更符合个人审美。

## 可用颜色设置的对象

| 图标类型 | 是否支持颜色 |
|----------|-------------|
| 内置 SVG 图标 | ✅ 支持 |
| FontAwesome 图标 | ✅ 支持 |
| Bootstrap Icons 图标 | ✅ 支持 |
| 导入的图片文件 | ❌ 不支持 |
| 导入的 SVG 代码 | ❌ 不支持 |
| Data URL 图片 | ❌ 不支持 |

## 设置颜色方法

1. 在图标选择器中选择图标
2. 点击颜色按钮（调色板图标）
3. 在颜色选择器中选择颜色
4. 图标会以选定颜色显示
5. 点击确认保存

## 颜色选择器功能

- **预设颜色**：常用颜色快速选择
- **色盘**：自定义颜色
- **透明度**：可设置半透明效果

## 清除颜色

如果想恢复默认颜色（白色）：

1. 点击颜色按钮
2. 选择透明/无颜色选项
3. 或点击清除颜色按钮

## 分类图标颜色

分类图标同样支持自定义颜色：

1. 选择分类
2. 点击图标区域
3. 选择图标后设置颜色
4. 确认保存
