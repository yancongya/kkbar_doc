# Bootstrap Icons

Kkbar 集成了 Bootstrap Icons 1.11.3，提供 300+ 个图标供选择。
![首页](/assets/images/iconinput.png)

## 使用方法

1. 在图标选择器中点击 **BI** 过滤按钮
2. 在搜索框中输入图标名称或关键词
3. 浏览搜索结果
4. 点击选择图标
5. 可选择设置图标颜色
6. 点击确认

**搜索技巧**：输入图标名称（如 `play-fill`、`save-fill`、`search`），Bootstrap Icons 提供常规版本（如 `heart`）和填充版本（如 `heart-fill`）两种变体。

## 常用图标

| 类别 | 图标名称示例 |
|------|-------------|
| 媒体 | play-fill, pause-fill, stop-fill, skip-backward, volume-up |
| 文件 | file-earmark, folder-fill, save-fill, printer, download |
| 编辑 | clipboard, clipboard-x, scissors, arrow-repeat, pencil |
| 箭头 | arrow-up, arrow-down, arrow-left, arrow-right |
| 常用 | gear, star-fill, heart-fill, trash, search |
| 天气 | sun, moon, cloud, snow, lightning |

## 核心特点

- **丰富图标**：300+ 个现代风格图标
- **双变体**：常规 + 填充两种风格可选
- **颜色自定义**：支持为图标设置自定义颜色
- **离线可用**：通过 CDN 加载，首次联网后缓存可用

## 适用场景

- 偏好 Bootstrap 风格的图标
- 需要填充/描边两种变体切换
- 与其他 Bootstrap 项目保持图标风格一致
