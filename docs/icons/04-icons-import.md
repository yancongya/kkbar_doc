# 04. 导入图标

除了内置图标和 FontAwesome，Kkbar 还支持导入自定义图片作为图标。

## 导入方式

### 方式一：导入图片文件

1. 在图标选择器中点击 **导入** 按钮
2. 在弹出的文件选择器中选择图片文件
3. 点击确认

**支持的图片格式**：

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| PNG | 支持透明背景 | Logo、复杂图标 |
| JPG/JPEG | 有损压缩 | 照片、复杂图像 |
| SVG | 矢量图形 | 矢量图标、Logo |
| GIF | 动态图标 | 简单动画效果 |
| WebP | 现代格式 | 高压缩比图像 |
| BMP | 位图格式 | 兼容性需求 |

### 方式二：粘贴 SVG 代码

如果已有 SVG 代码，可以直接粘贴导入：

1. 在图标选择器中点击 **粘贴** 按钮
2. 粘贴面板会展开
3. 在文本框中粘贴 SVG 代码
4. 输入图标名称（用于识别）
5. 点击 **保存 SVG** 按钮
6. 图标会出现在图标网格中

### 方式三：SVG 生成器（文字转图标）

如果只需要简单的文字图标，可以使用 SVG 生成器：

1. 在图标选择器中点击 **生成** 按钮
2. 生成器面板会展开
3. 输入文字（如 "AE"、"导出"、"P"）
4. 设置文字颜色（可选）
5. 设置背景颜色（可选）
6. 背景透明开关（可选）
7. 调整字体大小滑块
8. 实时预览效果
9. 点击 **生成 SVG** 按钮

**生成器选项说明**：

| 选项 | 说明 |
|------|------|
| 文字 | 图标上显示的文字（1-4 个字符效果最佳） |
| 文字颜色 | 文字的颜色 |
| 背景颜色 | 背景矩形的颜色 |
| 背景透明 | 开启后背景变为透明 |
| 字体大小 | 控制文字在图标中的比例（20-80px） |

**生成的 SVG 规格**：

- 画布尺寸：100×100 px（viewBox）
- 背景：圆角矩形（圆角半径 16px）
- 文字：水平居中垂直居中

## 导入的图标管理

导入的图标会保存到图标库中：

- 在图标选择器中点击 **导入** 标签
- 可以看到所有已导入的图标
- 右键点击图标可删除

## 图标存储位置

- **CEP 模式**：`{userData}/KkbarDB/icons/`
- **浏览器模式**：`localStorage`

## 核心特点

- **多种导入方式**：支持文件导入、粘贴 SVG 代码、文字生成三种方式
- **格式丰富**：支持 PNG、JPG、SVG、GIF、WebP、BMP 等多种图片格式
- **SVG 生成器**：内置文字转 SVG 图标功能，快速创建简单文字图标
- **颜色自定义**：生成器和导入的 SVG 支持设置颜色
- **图标管理**：导入的图标支持查看和删除
- **存储持久**：图标保存在本地，重启后仍可使用

## 适用场景

- 使用自己的 Logo 作为图标
- 使用项目专属的图标
- 使用复杂或特殊的矢量图标
- 快速创建文字图标（如首字母、功能简称）
- 导入现有的 SVG 素材作为图标
